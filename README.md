# scala-cli-sbt-crypto-bcrypt-hashing

## Description
A demo for Bcrypt one-way hashing of
a password.

## Tech stack
- sbt
- scala

## Docker stack
- openjdk:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
